Final Project

April 19, 2016

Final Project

  • Pick a data set. Some suggestions:
    • Lahman’s Baseball Database
    • US Airport Statistics
    • Moar US Airport Stats
    • ggplot2movies::movies built-in data set from the ggplot2movies packages (you’ll need to install that pkg)
    • Senate Bipartisan Rankings
    • Flint Water Data (you may want to hit up the Census web site as well)
  • Identify at least one question you want to answer using the data set
  • Use R to answer that/those question(s)

Minimum Requirements

  • Title
  • Question(s)
  • Data source(s) Reference(s) & Description
  • Basic descriptive statitstics (mean, medians, percentiles)
  • Salient exploratory data visualizations (can be simple bars, scatterplots, boxplots, lines)
  • Code you used to analyze the data
  • The answer(s) you found for the question(s)

Form

You will need to show all your work. There are two options for the output:

  • A Word/Google doc with the report text & graphics plus a separate file with R code
  • A “knitted” R Markdown report (covered today and April 27)
