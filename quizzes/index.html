Quiz 2

  1. Find the link to the county unemployment data file link on this course site

  2. Read it into R

  3. Find the county in Maine with the lowest unemployment and display the county name and unemployment rate

  4. Find the county in New Hampshire with the highest unemployment rate in 2014 and display the county name and unemployment rate

  5. BONUS: Take home, due by Saturday 23:59EST : for each state, display the counties with the highest and lowest unemployment rates. Output should be county name, state, rate with one row for highest and one for lowest for each state (104 total rows)

For both, save the output in your gbcc-rstudio server directory as quiz2.txt or quiz2.csv. You can do this by copying and pasting from the console to new source file and naming it quiz2.txt or use write.csv or write_csv from the tidyr package.

Hints:

  • tidyr, dplyr, readr will be useful packages
  • from tidyr, you may want to look at the help for separate
  • from dplyr, you may want to look at the help for arrange, filter, select, desc
  • you may want to refresh your memory of head and tail
